# Roomsphere
This project proposes the development of a centralized campus platform that streamlines roommate matching and move out product sales. The system will allow students to create detailed profiles and be matched with potential roommates using a transparent similarity-based scoring mechanism that considers lifestyle preferences, budget, and other compatibility factors, along with secure in-app chat for communication. In addition, the platform will include a structured marketplace where students can list and browse items for move-out sales, with smart search and filtering to improve product discovery and direct buyer–seller messaging for negotiation. The goal is to create a scalable, user-friendly solution that reduces reliance on fragmented social media groups and improves efficiency, trust, and convenience within the campus housing ecosystem.
